A Simple ZSH Config
===================

A simple configuration for getting started with zsh. 

Installation
------------

    cp .zshrc ~/.zshrc
    mkdir -p ~/bin
    cp bin/* ~/bin/
    chmod +x ~/bin/git-current-branch
    chmod +x ~/bin/git-cwd-info
